***i represents Number***

1. `i` -> Path index
2. `[i]` -> File index
3. `[/]` -> Represents current path (directory)
4. `$` -> sign of custom command

# Custom Commands `you have to start with [$]`

1. make {filename} `To make file`
2. makeDir {dir_name}
3. delete [i]
4. append , -r `to write somthing in a file. you can append or replace old text`
5. rename [i] {to name}
6. move
7. copy
8. watch
